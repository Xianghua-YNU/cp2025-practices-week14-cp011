### 3.3 实现过程中遇到的问题

请描述您在实现过程中遇到的主要问题及解决方法：

```
[1. 数值不稳定（种群数量出现负值或爆炸）
原因：

步长（dt）过大时，显式方法（如欧拉法）会放大误差。

模型非线性导致解对步长敏感。

解决方法：

减小步长：如从 dt=0.1 调整为 dt=0.01。

改用隐式方法：如后向欧拉法，但实现复杂。

限制种群范围：强制 x = max(x, 0) 和 y = max(y, 0)（物理意义补救）。

使用自适应步长：如RK45，动态调整步长（SciPy的solve_ivp内置支持）。]
```
